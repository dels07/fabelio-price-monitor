# Fabelio Price Monitor

## Description

A web application to scrap information and monitoring price from fabelio website.

## How to Install

1. Clone this repo
2. Cd into directory
3. Open up terminal
4. Type `composer install`
5. Type `npm install`
6. Type `php artisan serve`
7. Open `http://127.0.0.1:8000` in your browser
